# Chrome Web Store Resubmission Checklist

Use this checklist before resubmitting after an `Inaccurate Description - Irrelevant info` rejection.

## 1. Listing copy

- Keep the public listing language in Japanese.
- Make the first sentence describe the Chrome extension, not the website.
- Present the dashboard as a companion tool for managing saved profile data.
- Do not lead with resume parsing, dashboard sync, or generic recruiting-platform language.

## 2. Screenshots

Required rule:
- The first screenshot must clearly show the extension UI.

Recommended order:
1. Extension popup on a job application page
2. Autofill triggered on a real Japanese job application form
3. In-page Cygnet side panel on a real job application page
4. Dashboard profile editor as a companion screen
5. Optional resume manager or credential-vault screen

Do not upload screenshots that are primarily:
- a landing-page hero
- generic marketing art
- a website-only dashboard with no obvious extension context
- anything reviewers could interpret as unrelated to observed extension behavior

## 3. Promo image

Use a `440x280` promo tile that:
- shows the Cygnet name
- uses extension-first messaging
- uses popup/autofill visual cues
- avoids website-first imagery
- avoids small text or dense UI screenshots

Recommended direction:
- headline: `日本の応募フォームを自動入力`
- visual: popup plus a simplified job-form autofill motif

## 4. Reviewer notes

Include reviewer notes that explicitly state:
- Cygnet is a Chrome extension for autofilling Japanese job application forms
- the popup and page autofill are the core product behavior
- the dashboard is a companion interface for managing saved profile data
- the uploaded screenshots reflect the current extension functionality

Use:
- `/Users/maaku/Documents/Cygnet/docs/chrome-web-store-reviewer-notes.md`

## 5. Final pre-submit checks

- ZIP matches the current extension build
- screenshots match the current shipped UI
- store icon and promo tile use consistent branding
- first screenshot is extension-first
- no media looks like unrelated website or marketing material
- listing description and screenshots tell the same story
